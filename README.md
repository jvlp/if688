# Compiladores

## Task 1 - RPNStacker

entrada consiste em uma sequencia de números/operações separadas por uma quebra de linha. O caractere "q" encerra a execução

### Exemplo de input

5  
2  
+  
-3  
+  
2  
*  
4  
9  
-  
2  
/  
-  
q  

### output gerado
\>> 7  
\>> 4  
\>> 8  
\>> -5  
\>> -2  
\>> 10
